# Настройка и запуск сервера на Node.JS

## Подготовка

1. Установить [Node.JS](https://nodejs.org/en/download)
2. Для установки выбирать `prebuilt Node.js installer`
3. В ходе установки обязательно поставить галочку для добавления Node.JS в bin (или Path Variables)
4. Перезапустить ПК
5. Перейти в папку server (в этом репозитории), используя терминал (команда CD "путь")
6. Выполнить `npm install`
7. Выполнить `node exam.js`
8. Сервер запущен! В консоли вы сможете увидеть сообщение о запуске:
9. При написании тестов, можно использовать `localhost`, вместо IP-адреса.
